CSCE 242

Web Dev & Design

By: Christian Perez

In this class we are learning to create and design webpages that can later be used for potential business or personal usages. We learn the ins and outs of HTML and CSS in order to properly design webpages. This knowledge can be used over future prospects of working with a client who could potentially need a webpage made for them.

Assignments:

  1. Intro to HTML
  2. Intro to CSS
